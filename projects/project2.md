# 📊 Interactive COVID-19 Dashboard

**Objective:**  
Created a real-time dashboard to visualize global COVID-19 trends.

**Tools & Tech:**  
Python, Pandas, Plotly, Dash, REST API (COVID19 API)

**Highlights:**
- Connected to a live API to fetch daily cases, deaths, and vaccination data
- Built interactive charts (line graphs, choropleth maps) using Plotly Dash
- Filter by country and time range
- Hosted the app on Heroku with responsive UI
