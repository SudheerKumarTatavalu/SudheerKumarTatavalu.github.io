# 🧠 NLP Resume Classifier

**Objective:**  
Built a machine learning model to classify resumes into job categories (Data Science, Web Dev, etc.) using NLP techniques.

**Tools & Tech:**  
Python, Scikit-learn, NLTK, TF-IDF, Flask

**Highlights:**
- Preprocessed 500+ resumes with NLTK (cleaning, tokenization, stopwords removal)
- Engineered features using TF-IDF vectorization
- Trained and evaluated Logistic Regression and Random Forest classifiers (accuracy: 91%)
- Deployed a live web app using Flask to accept PDF uploads and classify the resume type
